# COS60008 Assignment Repository

This repository contains the starting files for a COS60008 Introduction to Data Science assignment submission.

## Repository Contents

- `COS60008_assignment_105974087.ipynb`: main working notebook
- `beach_amenities.csv`: beach amenity dataset
- `climate_summary_by_region.csv`: climate summary dataset
- `data_except_nsw.csv`: dataset excluding NSW
- `data_nsw.csv`: NSW dataset
- `data_dictionary.docx`: supporting data dictionary
